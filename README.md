# Mein Geld im Blick

Ein Geld-Planer in Einfacher Sprache. Die App läuft als statische Web-App/PWA und kann über GitHub Pages bereitgestellt werden.

## Was die App kann

- Einnahmen eintragen
- Ausgaben eintragen
- Kategorien auswählen
- Monatsübersicht anzeigen
- einfache grafische Übersicht mit Balken
- Ausgaben nach Kategorien anzeigen
- Daten lokal im Browser speichern
- CSV exportieren
- PDF-Übersicht über das Druckfenster erstellen
- JSON-Sicherung exportieren und wieder einlesen
- nach dem ersten Laden offline nutzen
- auf dem Handy zum Startbildschirm hinzufügen

## Datenschutz

Die Daten bleiben im Browser auf dem jeweiligen Gerät. Es gibt kein Konto und keine Cloud. Wenn Browserdaten gelöscht werden oder das Gerät verloren geht, können die Daten verloren sein. Deshalb ist der Export wichtig.

## GitHub Pages

1. ZIP-Datei entpacken.
2. Alle Dateien und Ordner in ein GitHub-Repository hochladen.
3. Wichtig: `index.html` muss direkt auf der obersten Ebene des Repositorys liegen.
4. In GitHub zu `Settings > Pages` gehen.
5. `Deploy from a branch` wählen.
6. Branch `main` und Ordner `/root` auswählen.
7. Speichern.

Danach zeigt GitHub eine Adresse an, zum Beispiel:

`https://BENUTZERNAME.github.io/REPOSITORY-NAME/`

## Nutzung auf dem Handy

### iPhone

1. Link in Safari öffnen.
2. Teilen-Symbol antippen.
3. „Zum Home-Bildschirm“ wählen.
4. Namen bestätigen.

### Android

1. Link in Chrome öffnen.
2. Menü mit drei Punkten öffnen.
3. „App installieren“ oder „Zum Startbildschirm hinzufügen“ wählen.

## Export

### CSV

Die CSV-Datei kann mit Excel, Numbers oder LibreOffice geöffnet werden.

### PDF

Die Schaltfläche „PDF-Übersicht erstellen“ öffnet eine Druckansicht. Dort kann man „Als PDF sichern“ oder „PDF speichern“ auswählen.

### Sicherung

Die JSON-Sicherung ist für die Wiederherstellung gedacht. Sie kann später wieder eingelesen werden.
