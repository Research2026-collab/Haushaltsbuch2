    Hilfe

    Wie kommt die App auf mein Handy?

    iPhone: Link in Safari öffnen. Auf Teilen tippen. Dann „Zum Home-Bildschirm“ wählen.

    Android: Link in Chrome öffnen. Menü öffnen. Dann „App installieren“ oder „Zum Startbildschirm hinzufügen“ wählen.

    Wo sind meine Daten?

    Die Daten werden nur im Browser auf diesem Gerät gespeichert. Niemand kann sie auf einem Server sehen.

    Was bedeutet Export?

    Export heißt: Sie speichern eine Datei mit Ihren Einträgen. Diese Datei können Sie behalten oder ausdrucken.
